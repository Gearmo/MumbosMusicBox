About/FAQ

About

Welcome to Mumbo's Musicbox! This is a music database website that holds music files compatible with the game "Banjo-Kazooie". This is mainly a quick-access tool for modders to find music for their levels, or for people simply searching for good video game music.

FAQ

Q) What file types are supported?
 A) Right now, our main files types we support are MP3, MIDI, and MCSZ files. These files are crucial to the development of BK modding, and helps everyone find the music regardless of musical knowledge.

Q) What is a MCSZ file?
 A) MSCZ files are music notation files used in MuseScore, which is one of the most popular notation software available for musicians. If you are interested in composing music for yourself, I would likely recommend using MuseScore. It has a relatively easy interface compared to other notation programs, and does not require a musical background to get started with.

Q) Why do your MIDI files sound so bad?
 A) MIDI files are the only compatible music files that can be imported into Banjo's Backpack, which is a modding program for editing levels, models, and music. Since Banjo-Kazooie's original music wasn't written in MIDI format, the in-game tracks and the MIDI files aren't 1-to-1. For example, the MIDI-track named "Flute" will not play a Flute sound in-game, but a track named "Harpsichord" will play back the sound of a Flute. Visit Composition tab for more information.
